## 信号总线 (EventBus) 隐式测试

所有信号总线交互应在相关功能测试中覆盖，不单独列出事件总线测试用例。需确保:

- [ ] 所有信号发送有对应连接
- [ ] 没有悬空连接 (对象释放时自动断开)
- [ ] 信号参数类型与连接函数参数类型匹配
