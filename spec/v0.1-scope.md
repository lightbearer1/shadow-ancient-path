# v0.1 范围 — 原型

## 概述

v0.1 是可玩原型，聚焦核心玩法: 玩家移动、战斗和基础敌人 AI。包含 1 个关卡 (3 个房间) 以验证核心循环。

---

## 范围

### 玩家系统

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 移动 | CharacterBody2D 水平移动, 加速度/减速度 | P0 |
| 跳跃 | 跳跃/二段跳跃, 可变跳跃高度 | P0 |
| 冲刺 | 方向冲刺, 无敌帧, 冷却 | P0 |

## 技术决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 引擎 | Godot 4.7 | 最新稳定版 |
| 玩家节点 | CharacterBody2D | 物理+碰撞检测 |
| 敌人节点 | CharacterBody2D | 统一物理接口 |
| UI 渲染 | CanvasLayer + Control | 不受屏幕抖动/相机影响 |
| 通信 | EventBus 信号单例 | 松耦合系统通信 |
| 关卡编辑 | 手工艺 TileMap | v0.1 范围小, 不需要程序生成 |
| 输入 | Input Map + Input.get_action_strength | Godot 标准输入系统 |
| 动画 | AnimationPlayer + AnimationTree | 状态驱动的动画切换 |
| 分辨率 | 640x360 (内部) -> 1280x720 (输出) | 像素艺术风格, 整数缩放 |
